# Semaphore

## Purpose 

Demonstrate the use of a semaphore with threads.

## File Structure
```
    |
    |-- include/
    |      + .h*
    |--- src/
    |      + .c*
    |
    + build.sh
    + CMakeLists.txt
```
## Requirements
- GCC
- Ubuntu 20.04 LTS
- General programming knowledge

## How to Build
1. Open Terminal
2. Type ```./build.sh```
3. Run executable in ```/build``` called "semaphore".
4. Exit
